Idea

MyMusic is a music player for Windows that aims to provide a lightweight solution for users with any performance level device.

My software uses the Windows UI to provide a simple and elegant user interface.

Thank you for your support! (/≧▽≦)/

UI Design

The UI design of MyMusic is based on Windows UI design style, which makes the interface simple and elegant, making the user feel comfortable and fluent. In the music playback page, besides the music player, there are also music lists, music playback speed, music reverberation, and music spectrum, providing various music playback functions, allowing users to find their favorite songs at any time and anywhere.

The software supports multiple DPI and Windows versions, which can adapt to various display screens, and supports zooming, allowing users to get the best visual experience on different screens.
